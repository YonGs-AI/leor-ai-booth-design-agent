# 安装说明

## MCP 地址

```text
https://leor.cn/agent-test/mcp
```

当前地址为公开测试入口。后续如 LEOR 切换正式 Agent 域名，请以 https://leor.cn/agent 公布的信息为准。

## 通用连接步骤

1. 打开你的 Agent 工具。
2. 找到 MCP、工具、插件、外部服务或自定义工具配置入口。
3. 新增一个 MCP 服务，名称建议填写：

```text
LEOR AI展台设计 Agent
```

4. MCP 地址填写：

```text
https://leor.cn/agent-test/mcp
```

5. 保存后，在 Agent 对话中输入：

```text
连接我的 LEOR 账号
```

6. 打开返回的 LEOR 授权链接，登录并确认授权。
7. 回到 Agent 对话继续使用。

## 支持工具

- Codex

- QClaw

其他支持 MCP 或外部工具调用的 Agent 环境，也可以按通用连接步骤尝试接入。

## 功能输入要求

- **新建展台设计**：可以从完整文字需求开始。Agent 必须先收集品牌、展品、宽度/深度/高度、前后左右墙体开口、风格、颜色、造型、材料/功能区/细节、模型、分辨率和画面比例，询问 Logo/参考图，然后汇总给用户确认。
- **改图**：必须有一张现有展台图和明确的修改要求。
- **万物成展**：必须有一张任意视觉来源，通常是动物、水果、产品、包装、Logo、色卡、材质或纹理，而不是现成展台图。
- **展位分镜/材质看板/海报/旋转视角**：必须有一张现有展台图。

用户可以在 Agent 对话中直接发送 JPG、PNG 或 WEBP 图片，单张不超过 25MB。不应要求普通用户自己找图床、复制图片 URL 或转换 Base64。

## 模型、分辨率和积分

| LEOR 公开模型名 | 1K | 2K | 4K |
| --- | ---: | ---: | ---: |
| 快速渲染模型 | 3 积分 | 4 积分 | 6 积分 |
| 专业渲染模型 | 4 积分 | 5 积分 | 7 积分 |
| 全能渲染模型 | 6 积分 | 8 积分 | 10 积分 |

默认使用 **专业渲染模型 + 2K**。生成 1 张新展台设计图通常消耗 5 积分。万物成展、展位分镜、材质看板和三类海报工具，在所选模型/分辨率的基础积分上加 3 积分。旋转视角也会扣积分。

## 常见问题

### 授权失败怎么办？

让 Agent 重新调用连接工具，或直接对 Agent 说：

```text
重新连接我的 LEOR 账号
```

不要让普通用户复制 token、查看 `.env`、检查本地源码或修改 MCP 服务代码。

### 图片一定要 URL 吗？

不一定。对于支持图片附件的 Agent，用户可以直接在对话中发送 JPG、PNG 或 WEBP 图片。Agent 应把图片作为当前任务输入交给 LEOR，不要要求普通用户自己找图床或转换 base64。

### 生成几张图？

LEOR Agent 默认每次只生成 1 张。若用户明确需要多张，Agent 也必须按单张任务依次执行，不开放批量并发；第一张返回后，再由用户确认是否继续下一张。
