# 抖音图文与短视频脚本

## 抖音图文标题

Agent 现在能直接做展台设计了

## 抖音图文正文

以前在 Agent 里聊完展台需求，还得自己打开生图页、上传 Logo 和参考图、把结果再放回项目。

这组图展示的是一条完整链路：在 Codex、Claude Code、OpenClaw、WorkBuddy、QClaw 这类 Agent 里连接 LEOR，先收集品牌、展品、尺寸、开口、风格、造型和材料，汇总确认后再生成 1 张展台设计图。

图片也可以直接发给 Agent，用来改图、做万物成展、材质看板、展位分镜或旋转视角。生成结果仍然保存在当前授权的 LEOR 项目里。

这种方式最值钱的，是少了在几个软件之间来回搬图。

你更想先让 Agent 帮你整理需求，还是直接改一张现有展台图？

## 45 秒口播

你平时用 Codex、Claude Code、OpenClaw 或 WorkBuddy，现在可以让它直接连接 LEOR 做展台设计了。

你说“我要做一个展台设计”，Agent 会先收集品牌名称、展示产品、宽度、深度、高度、前后左右的墙体开口，再问风格、颜色、造型、材料、Logo 和参考图。

资料齐了以后，它要先汇总给你确认，才能扣积分生成，而且默认只生成一张。

如果你发的是一张图，也不用自己找图床。直接说用它改图、做万物成展、材质看板或旋转视角就行。

最后出来的图，会继续保存在你的 LEOR 项目里。

## 镜头清单

1. `0-3s`：完整展台设计图，大字“Agent 现在能直接做展台设计了”。
2. `3-8s`：Agent 添加 LEOR MCP 的局部截图，不展示 token 或内部配置。
3. `8-16s`：对话中出现完整需求清单。
4. `16-22s`：用户上传 Logo 和参考图，Agent 汇总需求。
5. `22-27s`：LEOR 授权确认页，隐去账号隐私信息。
6. `27-34s`：展台设计图返回 Agent 对话。
7. `34-40s`：LEOR 项目内的主图、展位分镜、材质看板和旋转视角。
8. `40-45s`：字幕“在 Agent 里调用，在 LEOR 里继续保存”。

## 发布门禁

- 抖音文案和画面不出现 `leor.cn`、二维码和站外引流语。
- 如果画面或文字使用 AI 生成，后台勾选“内容由AI生成”。
- 不点击最终发布，除非勇哥明确批准本篇。
