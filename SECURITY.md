# 安全说明

LEOR AI展台设计 Agent 采用用户授权方式调用 LEOR。普通用户不需要复制 token，不需要查看本地配置文件，也不需要接触 API 密钥。

## 公开安全承诺

- Agent 调用前需要用户登录 LEOR 并确认授权。
- 扣积分功能消耗当前授权账号积分。
- 生成结果保存到当前授权账号的 LEOR 项目。
- LEOR 不向 Agent 暴露核心提示词、模型密钥、内部路由、后台路径或提示词资产。
- Agent/MCP 默认每次只生成 1 张，避免误触发批量扣费。
- 图片上传只能用于具体任务，不作为无限制网盘功能。

## 用户应该注意

- 只在你信任的 Agent 工具中添加 LEOR MCP 地址。
- 如果 Agent 要你复制 token、查看 `.env`、读取源码或修改本地配置，请停止并重新连接 LEOR。
- 如果 Agent 在资料明显不足时直接要生成图片，请先要求它汇总需求并确认。

## 安全反馈

如发现异常授权、异常扣费或 Agent 行为不符合说明，请通过 LEOR 官网联系团队：

https://leor.cn/
