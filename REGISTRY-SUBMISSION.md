# MCP 目录提交资料

## Name

LEOR AI展台设计 Agent

## Short Description

AI booth design agent connector for LEOR. It lets Codex, Claude Code, OpenClaw, WorkBuddy, QClaw, and other MCP-capable agents call LEOR to generate booth design images, edit booth images, create storyboards/material boards, and save results into the authorized LEOR account.

## 中文描述

LEOR AI展台设计 Agent 支持 Codex、Claude Code、OpenClaw、WorkBuddy、QClaw 等 Agent 工具，让 AI 通过对话调用 LEOR 完成展台设计、生图、改图和项目资产保存。

## Authentication

OAuth-style LEOR account authorization. Users log in to LEOR and confirm authorization. Normal users do not copy tokens manually.
