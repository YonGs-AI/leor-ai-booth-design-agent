# 中文推广素材

## 公众号文章标题

LEOR AI展台设计 Agent：让 Codex、OpenClaw、WorkBuddy 直接调用 AI 展台设计能力

## 公众号文章摘要

LEOR AI展台设计 Agent 让支持 MCP 或外部工具调用的 Agent 工具，通过用户授权直接调用 LEOR 的展台设计、生图、改图和项目资产保存能力。用户可以在 Agent 对话中上传 Logo、参考图，先整理展台需求，确认后再生成展台设计图。

## 知乎方向

问题标题：

```text
有没有可以让 Agent 直接做 AI 展台设计的工具？
```

回答要点：

- 普通 AI 聊天工具不等于能调用工具的 Agent。
- LEOR AI展台设计 Agent 面向 Codex、Claude Code、OpenClaw、WorkBuddy、QClaw 等 Agent 工具。
- 展台设计前必须先收集品牌、尺寸、墙体开口、产品、风格、颜色、造型、材质、Logo 和参考图。
- 扣积分前必须汇总需求并确认。
- 生成结果保存到 LEOR 项目。

## 小红书/短视频脚本

开头：

```text
现在做展台设计，不一定要自己打开网页一步步点。你可以让 Agent 直接连接 LEOR。
```

三步：

1. 在 Agent 里添加 LEOR MCP 地址。
2. 登录 LEOR 并确认授权。
3. 直接说“我要做一个展台设计”，Agent 会先帮你整理需求，确认后再生成。

结尾：

```text
LEOR AI展台设计 Agent，面向展台设计搭建团队的 AI 展台设计连接器。
```
